# You May Know Recommendation System

## Description
This is a simple Python project that suggests people you may know based on mutual friends.

## Features
- Suggests new connections
- Uses mutual friend logic
- Filters existing friends

## Tech Used
- Python

## How to Run
1. Download the file
2. Run this command:
   python you_may_know.py

## Example
Enter your name: Rupak  
People you may know: ['Riya', 'Rahul']
